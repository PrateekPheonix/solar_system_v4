Instructions how to navigate by the simulator

Mouse button - Use buttons to rotate by the universe :)
Mouse wheel - Use the wheel to zoom in or zoom out.

Keyboard - Type "R" to reset the camera view
Keyboard - Type "P" to Pause or Play animation. When on Pause you can rotate or zoom the scene.

Keyboard - Type "." to increase the orbit speed of the planets
Keyboard - Type "," to decrease the orbit speed of the planets

Scene Axis - Axis Red,Green and blue represents Axis X, Y and Z
Keyboard - Type "O" to play or pause the Orbits of the planets

Keyboard - Type "M" to focus on Mercury. try to keep pressing "M" to follow its movement
Keyboard - Type "V" to focus on Venus
Keyboard - Type "E" to focus on Earth.
Keyboard - Type "A" to focus on Mars.
Keyboard - Type "J" to focus on Jupyter.
Keyboard - Type "S" to focus on Saturn.
Keyboard - Type "U" to focus on Uranus.
Keyboard - Type "N" to focus on Neptuno.
